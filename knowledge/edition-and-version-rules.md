# Bedrock edition and version rules

Minecraft Java Edition and Bedrock Edition share many ideas but differ in redstone behavior, commands, block identifiers, add-on APIs, update timing, and multiplayer servers. An answer should name the edition and should not silently translate a Java recipe into Bedrock.

Preview and beta notes describe upcoming behavior. They are not evidence that a feature exists in the current stable game. Normal questions should retrieve stable documentation first. Preview material should appear only when the player explicitly asks about preview, beta, or experimental behavior.

Patch notes record what changed in a particular release; they are historical events, not always the best description of current behavior. Prefer current stable reference documentation and tested mechanic cards, then use patch notes to explain when or why something changed.
